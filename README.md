## Общий функционал
Приложение, которое позволяет пользователю войти в чат-комнату под выбранным
никнеймом, просматривать историю сообщений, получать новые сообщения и отправлять
сообщения. Все сообщения - текстовые.

## Основные технологии
* Python
* Django
* Docker Compose
